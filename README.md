# Hotel Feedback Form

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic hotel feedback form webpage created using HTML. It allows users to enter personal information, answer questions about their stay, select reasons for choosing the hotel, rate services, and leave additional comments.

The page was built to practise creating structured and interactive web forms using standard HTML form elements.

## Features

- Page header with title and introduction
- Form with multiple fieldsets for clear organization
- Text, email, and number input fields
- Radio buttons for single-choice questions
- Checkboxes for multiple-choice answers
- Dropdown menus for service and food ratings
- Textarea for additional comments
- Submit button for form submission

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Building forms using the `form` element
- Grouping related inputs with `fieldset` and `legend`
- Using labels for accessibility
- Working with text, email, and number inputs
- Using radio buttons and checkboxes
- Creating dropdown menus with `select` and `option`
- Adding a textarea for user comments
- Structuring form content clearly and logically
